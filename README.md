# partial-clustering-ssizecalc
R code for the partial clustering sample size app at https://klange.shinyapps.io/partial-clustering-ssizecalc/, a sample size calculator for partially clustered trials. An accompanying tutorial manuscript is published at:

Lange KM, Sullivan TR, Kasza J, Yelland LN. An online sample size calculator for partially clustered trials. Online ahead of press, Clinical Trials 30 Jan 2026. https://doi.org/10.1177/17407745251415536
